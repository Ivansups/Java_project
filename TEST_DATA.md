## Примечания

- Все данные создаются автоматически при первом запуске приложения
- Если база данных уже существует, тестовые данные не добавляются повторно
- Для очистки и пересоздания данных удалите файл `library.db` и перезапустите приложение
- Читатели и пользователи - это разные сущности:
  - **Пользователи (users)** - для входа в систему (логин/пароль)
  - **Читатели (readers)** - клиенты библиотеки (ФИО, номер билета, контакты)
